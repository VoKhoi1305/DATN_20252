## Props

| Prop     | Type      | Default   | Mô tả                                            |
|----------|-----------|---------|---------------------------------------------------|
| children | ReactNode | —       | Nội dung form (Login form, OTP form, Setup form)  |
| title    | string    | undefined | Tiêu đề phía trên form (VD: "Xác thực OTP")     |
| subtitle | string    | undefined | Mô tả ngắn dưới tiêu đề                          |

| State   | Mô tả                                              |

## Behavior
- Panel trái là static branding — không thay đổi giữa các màn auth

- Form container: `max-w-[380px] w-full` — center trong panel phải
- `title` và `subtitle` render phía trên children trong panel phải
- Footer cố định dưới cùng

## Accessibility
- `<main>` cho panel phải
- `role="banner"` cho panel trái
- Focus auto vào field đầu tiên khi trang load

# A.2 — COMPONENT: Button (CMP-BTN)

## Mô tả
Nút hành động dùng cho mọi tương tác. Có 5 variants phân biệt theo mức độ quan trọng. Mỗi khu vực UI chỉ có tối đa 1 nút `primary`.

## Variants
primary | secondary | outline | ghost | danger-ghost

## Khi nào dùng variant nào (trong context OTP screens)
```
primary      → "Xác thực" (SCR-002), "Xác nhận & Lưu" (SCR-003)
ghost        → "Quay lại", "Dùng backup code"
outline      → "Sao chép" secret key
```

## Sizes

| Size | Height | Padding X | Font  | Tailwind                     | Dùng cho                    |
|------|--------|-----------|-------|------------------------------|-----------------------------|
| sm   | 30px   | 10px      | 12px  | h-[30px] px-[10px] text-xs   | Link-like actions            |
| md   | 36px   | 14px      | 13px  | h-9 px-[14px] text-[13px]    | Default — form, card, modal |
| lg   | 40px   | 20px      | 13px  | h-10 px-5 text-[13px]        | Form submit chính, CTA lớn  |

## Props

| Prop       | Type                                                           | Default     | Mô tả                              |
|------------|----------------------------------------------------------------|-------------|-------------------------------------|
| variant    | 'primary' \| 'secondary' \| 'outline' \| 'ghost' \| 'danger-ghost' | 'primary'   | Visual variant                      |
| size       | 'sm' \| 'md' \| 'lg'                                          | 'md'        | Kích thước                          |
| disabled   | boolean                                                        | false       | Vô hiệu hoá                         |
| loading    | boolean                                                        | false       | Hiện spinner, tự disabled           |
| fullWidth  | boolean                                                        | false       | width 100%                          |
| leftIcon   | ReactNode                                                      | undefined   | Icon bên trái text                  |
| rightIcon  | ReactNode                                                      | undefined   | Icon bên phải text                  |
| onClick    | () => void                                                     | —           | Handler click                       |
| type       | 'button' \| 'submit' \| 'reset'                               | 'button'    | HTML button type                    |
| children   | ReactNode                                                      | —           | Nội dung nút                        |
| className  | string                                                         | undefined   | Custom classes bổ sung              |

## Layout

```
[leftIcon?] [spinner? | children] [rightIcon?]
  gap-2 giữa icon và text
  spinner thay thế leftIcon khi loading (giữ text)
```

## Behavior
- `loading=true` → hiện spinner bên trái (thay leftIcon), giữ text, `disabled=true` tự động
- `fullWidth=true` → `w-full`
- Chỉ 1 nút `primary` trên mỗi khu vực logic
- Spinner element: `<span className="h-3 w-3 border-2 border-current border-t-transparent rounded-full animate-spin" />`

## Accessibility
- `aria-busy="true"` khi loading
- `aria-disabled="true"` khi disabled
- Enter + Space trigger onClick
- Focus visible bằng ring (không dùng outline)

## Props

| Prop        | Type                    | Default   | Mô tả                                       |
|-------------|-------------------------|-----------|----------------------------------------------|

## Layout

```
┌─────────────────────────────────────────────────────┐
│ [Label text] [* đỏ nếu required]                    │  ← text-xs font-medium text-zinc-800
├─────────────────────────────────────────────────────┤
│ [leftIcon?] │ input text                 │ [rightIcon?] │  ← h-9 (36px), border zinc-300, rounded
├─────────────────────────────────────────────────────┤
│ [error text HOẶC helper text]                        │  ← text-xs mt-1
└─────────────────────────────────────────────────────┘

Khoảng cách:
  Label → Input:    gap-1 (4px)
  Input → Helper:   mt-1 (4px)
  leftIcon:         pl-9 cho input (icon absolute left-3)
  rightIcon:        pr-9 cho input (icon absolute right-3)
  Icon size:        16x16 (h-4 w-4), color zinc-400
```

## Behavior
- `error` có giá trị → border red-700, bg red-50, hiện error text (thay helper text)
- `required=true` → dấu `*` màu `text-red-700 ml-0.5` bên cạnh label
- `monospace=true` → input text chuyển sang `font-mono text-[12px] tracking-wide text-zinc-600`
- Label và input liên kết qua `htmlFor` / `id`
- Error text và helper text KHÔNG hiện đồng thời — error ưu tiên hơn

## Accessibility
- `<label htmlFor={id}>` liên kết input qua id
- `aria-invalid="true"` khi có error
- `aria-describedby={helperTextId | errorTextId}`
- `aria-required="true"` khi required
- Tab navigation focus vào input

## API Integration

### Xác thực OTP — POST /api/v1/auth/verify-otp

- Trigger: Auto-submit khi đủ 6 ký tự HOẶC click nút "Xác thực"
- Request headers:
  ```
  Authorization: Bearer {tempToken}
  Content-Type: application/json
  ```
- Request body:
  ```json
  {
    "otp_code": "123456"
  }
  ```
- Response 200:
  ```json
  {
    "access_token": "eyJhbG...",
    "refresh_token": "eyJhbG...",
    "user": {
      "id": "uuid",
      "name": "Nguyễn Văn A",
      "role": "CAN_BO_CO_SO",
      "data_scope": "Phường Bến Nghé, Q.1",
      "otp_enabled": true
    }
  }
  ```
  - Action: Lưu access_token + refresh_token vào global state (Zustand)
  - Action: Lưu user info vào auth state
  - Navigate: /dashboard bằng router.replace
  - Nếu có returnUrl trong query params → navigate đến returnUrl
- Response errors:

  | Status | Code                | Message hiển thị                                          | Xử lý thêm                          |
  |--------|---------------------|-----------------------------------------------------------|--------------------------------------|
  | 400    | INVALID_OTP         | "Mã OTP không đúng. Vui lòng thử lại."                   | Clear OTP input, focus ô đầu, shake animation |
  | 400    | OTP_EXPIRED         | "Mã OTP đã hết hạn. Vui lòng nhập mã mới."              | Clear OTP input, reset countdown     |
  | 401    | SESSION_EXPIRED     | "Phiên đăng nhập đã hết hạn. Vui lòng đăng nhập lại."   | Redirect /login                      |
  | 429    | TOO_MANY_ATTEMPTS   | "Quá nhiều lần thử sai. Vui lòng đợi 5 phút."           | Disable form, hiện countdown 5 phút  |
  | 500    | SERVER_ERROR        | "Lỗi hệ thống. Vui lòng thử lại sau."                   | Hiện Toast error                     |

## Form Fields

### OTP Code (Step 2 — xác nhận)
- Component: CMP-INPUT (OTP variant — 6 ô riêng lẻ)
- type: text (inputmode="numeric" pattern="[0-9]*")
- monospace: true
- maxLength: 6
- autoFocus: true (khi chuyển sang Step 2)
- validation:
  - required → "Vui lòng nhập mã OTP"
  - pattern (chỉ số 0-9) → "Mã OTP chỉ chứa số"
  - length === 6 → "Mã OTP phải đủ 6 số"
- behavior: Giống OTP input ở SCR-002

### Checkbox xác nhận (Modal Backup Codes)
- Component: native checkbox + label
- label: "Tôi đã lưu các mã dự phòng ở nơi an toàn"
- checked: boolean (local state)
- behavior: Khi checked → enable nút "Đã lưu, tiếp tục"

## API Integration

### Lấy QR Code & Secret Key — GET /api/v1/auth/setup-otp

- Trigger: Page load (khi mount component)
- Request headers:
  ```
  Authorization: Bearer {tempToken}
  ```
- Response 200:
  ```json
  {
    "qr_code_uri": "otpauth://totp/SMTTS:nguyen.van.a?secret=JBSWY3DPEHPK3PXP&issuer=SMTTS",
    "secret_key": "JBSW Y3DP EHPK 3PXP",
    "account_name": "nguyen.van.a"
  }
  ```
  - Action: Render QR code từ `qr_code_uri` (dùng thư viện qrcode hoặc nhận base64 image)
  - Action: Hiển thị `secret_key` dạng text monospace
- Response errors:

  | Status | Code              | Message hiển thị                                  | Xử lý                      |
  |--------|-------------------|---------------------------------------------------|-----------------------------|
  | 401    | SESSION_EXPIRED   | "Phiên đăng nhập đã hết hạn."                    | Redirect /login             |
  | 409    | OTP_ALREADY_SETUP | "OTP đã được cài đặt cho tài khoản này."         | Redirect /login/otp         |
  | 500    | SERVER_ERROR      | "Lỗi hệ thống. Vui lòng thử lại sau."           | Hiện error state + nút retry |

### Xác nhận OTP & Lưu — POST /api/v1/auth/confirm-otp-setup

- Trigger: Click nút "Xác nhận & Lưu" hoặc auto-submit khi đủ 6 ký tự
- Request headers:
  ```
  Authorization: Bearer {tempToken}
  Content-Type: application/json
  ```
- Request body:
  ```json
  {
    "otp_code": "123456",
    "secret_key": "JBSWY3DPEHPK3PXP"
  }
  ```
- Response 200:
  ```json
  {
    "success": true,
    "backup_codes": [
      "AB12-CD34", "EF56-GH78", "IJ90-KL12", "MN34-OP56", "QR78-ST90",
      "QR56-ST78", "UV90-WX12", "YZ34-AB56", "CD78-EF90", "GH12-IJ34"
    ],
    "access_token": "eyJhbG...",
    "refresh_token": "eyJhbG...",
    "user": {
      "id": "uuid",
      "name": "Nguyễn Văn A",
      "role": "CAN_BO_CO_SO",
      "data_scope": "Phường Bến Nghé, Q.1",
      "otp_enabled": true
    }
  }
  ```
  - Action: Mở Modal backup codes (CMP-MODAL) với danh sách backup_codes
  - Action: Lưu token TẠM (chưa navigate, chờ user lưu backup codes)
- Response errors:

  | Status | Code                | Message hiển thị                                  | Xử lý                                 |
  |--------|---------------------|---------------------------------------------------|----------------------------------------|
  | 400    | INVALID_OTP         | "Mã OTP không đúng. Kiểm tra lại app."          | Clear input, shake, re-focus           |
  | 400    | OTP_EXPIRED         | "Mã OTP đã hết hạn. Chờ mã mới."               | Clear input, re-focus                  |
  | 401    | SESSION_EXPIRED     | "Phiên đăng nhập đã hết hạn."                   | Redirect /login                        |
  | 409    | OTP_ALREADY_SETUP   | "OTP đã được cài đặt."                          | Redirect /login/otp                    |
  | 429    | TOO_MANY_ATTEMPTS   | "Quá nhiều lần thử. Đợi 5 phút."               | Disable form, countdown 5 phút         |
  | 500    | SERVER_ERROR        | "Lỗi hệ thống. Vui lòng thử lại sau."          | Toast error                            |

## Edge Cases

| Case                                      | Xử lý                                                     |
|-------------------------------------------|------------------------------------------------------------|
| User truy cập /setup-otp khi đã setup OTP | API trả 409 → redirect /login/otp                         |
| User truy cập /setup-otp không có session  | Redirect /login                                            |
| QR code không load được                   | Hiện nút "Thử lại" + hướng dẫn nhập secret key thủ công  |
| User copy secret key                      | Toast success + nút đổi thành "Đã sao chép ✓" 2 giây     |
| User copy backup codes                    | Toast success "Đã sao chép 10 mã vào clipboard"           |
| User tắt/refresh tab khi đang ở Step 3    | Backup codes MẤT (không thể hiện lại). User cần Admin reset OTP. Hiện warning trong modal. |
| User chưa check "Tôi đã lưu" mà click nút | Nút disabled, hiện tooltip "Vui lòng xác nhận đã lưu mã" |
| OTP verify sai → quay lại Step 1          | Secret key và QR KHÔNG thay đổi (cùng session)            |
| Authenticator app timezone sai            | Mã OTP có window ±30 giây (server config)                 |
| Mạng mất khi load QR                      | Toast "Không có kết nối mạng" + nút Thử lại              |
| Admin reset OTP → user đăng nhập lại      | Redirect /setup-otp tự động                               |
